# FusionPulse CLI

Generate and run Playwright E2E tests from plain English.

```bash
npm install -g @fusionpulse/cli

# Describe a test in English
fusionpulse generate "Test the login flow: enter email and password, click submit, verify dashboard"

# Run tests
fusionpulse run

# Record browser interactions
fusionpulse record

# Initialize a project
fusionpulse init
```

[Learn more at fusionpulse.colefusion.net](https://fusionpulse.colefusion.net)
